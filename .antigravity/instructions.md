# Objetivo del proyecto

Crear una aplicación web de sistema de asistencias para una institución educativa pública superior, UBBJ en México. Registra entradas, calcula puntualidad y retardos, genera reportes y estadísticas para supervisión administrativa (en página web con login).

Se usará FastAPI, PostgreSQL y React (React Router + JavaScript + Vite) y Bootstrap.

El proyecto ya está creado, pero sin estilos ni funcionalidades. Hay dos objetivos, uno es implementar el diseño (frontend) y las funcionalidades básicas de la API, con un diseño premium y minimalista, siguiendo "casi" exactamente los diseños presentados en las referencias que serán proporcionadas que se encuentran en la carpeta ".antigravity/resources/" dentro del proyecto.

El otro objetivo es implementar las funciones de generación de reportes y estadísticas en el backend, y crear una plantilla de Excel base, para subir este mismo archivo .xlsx desde el frontend y cargar las asistencias en el backend.
