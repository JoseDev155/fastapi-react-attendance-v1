# Especificación del Template de Asistencia (V2)

## 1. Archivos Base

* **Plantilla Maestra:** `Plantilla.xlsm` (Libro habilitado para macros).
* **Lógica VBA:** El código unificado reside en `XlsmExpress.vba` (Evento Worksheet_Change).

## 2. Funcionamiento del Excel

* **Hoja 'Asistencias YYYY-MM'**:
  - Las celdas de asistencia tienen un `0` inicial (formato invisible).
  - Al ingresar `1`, la macro captura `Time` (hora actual) y lo guarda en la celda.
  - El backend debe interpretar cualquier valor de tiempo como `PRESENT` o `LATE` (comparando contra el horario de clase).
* **Hoja 'Asistencia Express'**:
  - **C6 (Trigger)**: Al poner `1`, aplica asistencia masiva usando el día en `A6` y la hora en `B6`.
  - **E6 (Configuración)**: Filtra visualmente las columnas de la hoja principal (ej: "Lunes, Jueves").
* **Identificación Unívoca**: El Excel DEBE contener una columna oculta (Columna A o AI, preferiblemente AI) con el `enrollment_id` o `student_ID`. El mapeo por nombre es solo un respaldo (fallback).

## 3. Esquema de Base de Datos Relacionado

El backend debe procesar el Excel y realizar un UPSERT en:

```pgsql
CREATE TABLE attendances (
    id INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    attendance_date DATE NOT NULL,
    arrival_time TIME, -- IMPORTANTE: Captura la hora exacta del Excel
    notes VARCHAR(255),
    enrollment_id INTEGER REFERENCES enrollments(id) NOT NULL,
    UNIQUE(enrollment_id, attendance_date)
);
```

## 4. Instrucciones para la Generación Dinámica

Cuando el usuario solicite la descarga de una plantilla:

1. **Fecha Dinámica** : Obtener mes/año actual y reemplazar el marcador `- MES, AÑO` en el encabezado.
2. **Calendario** : Generar los encabezados (ej: "Lun 01", "Mar 02") basados en los días reales del mes solicitado.
3. **Inyección de Macros** : Utilizar `xlsxwriter` en Python para insertar el archivo de macros binario (`vbaProject.bin`) en el nuevo archivo generado, asegurando que la extensión sea `.xlsm`.
4. **Renombrado de Referencias** : El agente debe asegurarse de que el código VBA interno haga referencia al nombre dinámico de la hoja (ej: "Asistencias 2026-04").
