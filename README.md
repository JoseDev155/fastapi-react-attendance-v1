Registra entradas, calcula puntualidad y retardos, genera reportes y estadísticas para supervisión administrativa.

## Tecnologías Utilizadas

* **IA**: Google Antigravity, Google Gemini, Google Stitch, Vibecoding (mayormente para el frontend, pero para una versión de prueba para los módulos de reporte y estadísticas del backend)
* **Backend**: FastAPI (Python)
* **Base de Datos**: PostgreSQL
* **Frontend**: React.js, Bootstrap
* **Autenticación**: JWT (JSON Web Tokens)

Enlaces a las herramientas de IA:

* [Google Antigravity](https://antigravity.google/)
* [Google Stitch](https://stitch.withgoogle.com/)

>Tengo suscripción Pro a las IAs de Google, pero Google es bastante generoso con los créditos en Stitch ya que está en BETA, por lo que no debería haber problemas en usarlo

>En el caso de Antigravity, también es generoso con las peticiones Pro iniciales, por lo que debería dar suficiente para 1-2 proyectos completos antes que se agoten las peticiones Pro
